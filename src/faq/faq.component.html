FAQ / Troubleshooting

For general troubleshooting or if your question is not answered here please join the OverTrack Discord.

Is OverTrack a cheat?

Blizzard's current stance on third party programs is unclear. Some third party programs with similar functionality to OverTrack have been forbidden, although no bans or warnings have been reported stemming from the use of OverTrack.

OverTrack behaves just like any other screen recording software e.g. OBS, Shadowplay. It does not interact with Overwatch and does not inject, modify or read Overwatch's memory, code or data files.
Data is only uploaded after a game has completed, so OverTrack has no way of providing an advantage while in game.


Where does OverTrack install to?

OverTrack installs to "%appdata%\overtrack" and should be available under all programs > OverTrack launcher.
To stop using OverTrack, you can remove it through "Add or remove programs" or just disable 'Launch on Startup' in the Settings menu in from the taskbar.


OverTrack is not picking up Overwatch

OverTrack can have difficulty picking up Overwatch for the following reasons:

  • Overwatch is being run as admin, but OverTrack is not.
  • Overwatch is being run in fullscreen mode on Windows 7. Windows 7 fullscreen capture requires using OBS Capture.
  • You are using a laptop with switchable graphics and running Overwatch on the dedicated GPU. Laptop fullscreen capture requires using OBS Capture.
  • Overwatch is not being run in 16:9 aspect aspect ratio (ingame setting).
  • You have an in-game overlay (e.g. ShadowPlay overlay, Discord Overlay, Windows registration text). Either disable this overlay or use OBS Capture.
  • Overwatch is not set to English.
  • The killfeed is disabled.

How does OverTrack work?

OverTrack applies Computer Vision and Machine Learning techniques to analyse and extract information from what is displayed onscreen. This allows OverTrack to track information such as match SR, map, killfeed and tab-screen statistics not directly provided by Blizzard.


OBS Integration

To use OBS capture mode will need to run OBS alongside OverTrack, with a plugin installed allowing OverTrack to use OBS's capture method.

Download obs-screenshot-plugin.dll from the releases page and install it to C:\Program Files\obs-studio\obs-plugins\64bit
Relaunch OBS and ensure you have a source for your game (game capture or capture card) then right click your target source and select Filters then add a Screenshot Filter:

Select the Destination type Output to Named Shared Memory and, the Shared Memory Name to overtrack:overwatch or overtrack(for autoselecting game) and the Interval to 1.0

If the source is a display capture, capture card, or game capture for "any fullscreen application" and you used the name overtrack and OverTrack will attempt to automatically detect the game. It does this by checking for the main menu periodically. You may need to wait at most 5s in the main menu for OverTrack to pick up the OBS source fully. Once seen, OverTrack should show that the source is Shared Memory in the Status Window. If the source is a game capture for a specific game and using the name overtrack:overwatch it will override the autodetection.
Do not create multiple filters with the same shared memory name.

To check if OBS capture is working, enable the Status Window and look for the text OBS shmem in the image, or SharedMemorySource in the text.

OverTrack is causing stutter/frame drops

Stutter/frame drops can be caused by OverTrack's fullsceen capture method - specifically if you are using multiple monitors and playing media on a secondary monitor while playing Overwatch, although it can also occur without this.
Using OBS Capture will mitigate this issue.


OverTrack is missing my SR for games

In order to track SR, make sure you are visiting the play menu before and after each game so OverTrack can capture the SR. If OverTrack missed the SR for a game you can manually provide it using the 'Edit' button on games.


I have a game showing up under the wrong account/in another tab in the games list

Sometimes OverTrack can have trouble working out the name of a player that a game belongs to. This can happen if you were alt-tabbed out when the game started. To fix games with the wrong player name you can use the 'Edit' button to change the account name a game is associated with.


My games/some of my games show up under an account beginning with @

OverTrack is unable to read special characters in player names - if your account has special characters your games will show up under '@battletag'.
You can also let OverTrack use the battletag from your currently logged in Battle.net account by enabling "Read Battletag From Blizzard App"
. If your account does not have special characters this could be caused by OverTrack having difficulty detecting the correct player name. Use the 'Edit' button to correct these games, and report the issue through discord if this happens frequently.


Why are some players @UNKNONWN_BLUE#0 or @UNKNONWN_RED#1?

@UNKNOWN names appear for players who use special characters as part of their battletags.
